1. Clone Repository
git clone https://github.com/vuchau0802/Sentiment-Analysis-Mini-Project.git
cd Sentiment-Analysis-Mini-Project

2. Create Virtual Environment
python -m venv venv
venv\Scripts\activate  (Windows)
source venv/bin/activate (Mac/Linux)

3. Install Requirements
pip install -r requirements.txt

4. Download NLTK Dataset
python
import nltk
nltk.download('vader_lexicon')
exit()

5. Run Application
python app.py

6. Open in Browser
http://127.0.0.1:5000
